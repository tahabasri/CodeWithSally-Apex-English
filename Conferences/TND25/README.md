## ðŸ›  How to Use the Snippets in VS Code

You can load the `agentforceDev_Prompts.code-snippets` into your VS Code environment and start using them right away.

### ðŸ’» On macOS

1. Press `Command + Shift + P` to open the **Command Palette**.
2. Type and select: `Snippets: Configure Snippets`.
3. Choose one of the following:
   - Select an **existing global snippets file**, or
   - Choose **New Global Snippets file** or **New Snippets file for your project**.
4. Paste the contents of `agentforceDev_Prompts.code-snippets` into the file.
5. Save (`Cmd + S`) and test it out in your project.

### ðŸªŸ On Windows

1. Press `Ctrl + Shift + P` to open the **Command Palette**.
2. Type and select: `Snippets: Configure Snippet`.
3. Choose one of the following:
   - Use an **existing snippet file**, or
   - Choose **New Global Snippets file** or **New Snippets file for your project**.
4. Paste the contents of `agentforceDev_Prompts.code-snippets`.
5. Save (`Ctrl + S`) and try out your new snippets!
