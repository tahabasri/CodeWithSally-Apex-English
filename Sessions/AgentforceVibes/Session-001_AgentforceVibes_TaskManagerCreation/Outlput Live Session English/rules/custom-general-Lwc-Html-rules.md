# Rule: Code With Sally LWC HTML Best Practices

**Description:** Enforce some best practices and naming and formatting standards for Lwc html code

**Applies to:** `**/*.html` 

**Guidelines:**
- Prioritize Lightning base components and SLDS utilities, minimizing raw HTML. (for example: `lightning-button-icon`, `lightning-layout`, `lightning-layout-item`, etc.).
- For styling, always prefer SLDS classes before creating custom CSS classes.
- Include a comment block before each major section of the template.
