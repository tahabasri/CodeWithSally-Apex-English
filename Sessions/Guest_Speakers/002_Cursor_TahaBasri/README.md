# Guest Speaker: Cursor - Taha Basri
This is the repository for the guest speaker session with Taha Basri from Cursor. In this session, Taha discusses the features and benefits of using Cursor for coding and development. All assets in current folder are referenced in the main video located at: https://www.youtube.com/watch?v=YCEuao46Cto

Note:
The assets that were generated in the English version were mistakenly deleted after the recording. The files referenced here are from the Arabic version of the session, but they are identical in content to those used in the English version.

# What's Included
- `Code with Sally - Cursor - English.pdf` — Presentation slides used during the live session, in English.
- `.cursor/` — Cursor-specific metadata and configuration files used in the session.
- `cursor_get_number_of_task_records.md` — Chat extract from Cursor showing how to get the number of Task records in a Salesforce org using Apex.
- `cursor_implement_requirements_and_await.md` — Implementation notes showing how requirements were implemented.
- `delta-package.xml` — Metadata package fragment useful for deploying the included components to a Salesforce org, generated by custom command from Cursor.
- `force-app/` — Salesforce metadata (Apex classes, LWC, and other components) used or referenced in the session.
- `scripts/` — Helper scripts used during the session for building, packaging, or deploying the sample code.

Notes:
- Review the markdown files for step-by-step explanations and code snippets used in the live session video.
- Use the `delta-package.xml` together with the `force-app` folder when deploying to a target org.
